Grounding Before Generalizing: How AI Differs from Humans in Causal Transfer

四月 28, 2026 2-minute read
Paper-Digest • 2026-04-28

arXiv: 2604.24062 · PDF

作者: Liangru Xiang, Yuxi Ma, Zhihao Cao, Yixin Zhu, Song-Chun Zhu

单位: Tsinghua University, Peking University, State Key Laboratory of General Artificial Intelligence

主分类: cs.AI · 全部: cs.AI

命中关键词: large language model, llm, agent, rag, reasoning

TL;DR

在 OpenLock 因果结构迁移任务上，LLM/VLM 必须先做环境特定的 grounding 才能获得迁移收益，而人类从第一次尝试就能用上先验结构；视觉信息反而拖累表现。

Motivation

抽取抽象因果结构并迁移到新场景是人类智能的标志，但当前 LLM/VLM 是否具备这一能力没有被系统刻画过。经典 RL agent 在这类任务上灾难性失败，人类却能在极少暴露后完成迁移，这中间到底隔了什么？对 agent runtime 团队和做 long-horizon 规划 / 工具使用的开发者来说，这是一个现实的痛点：今天的 LLM agent 在一个环境里调通之后，换一个物理布局、换一批 lever 标签，往往又要从零摸索，复用先前"学到的因果结构"的能力被悄悄高估。作者选择用 OpenLock 这一经典人机对比范式，正是因为它要求顺序发现 Common Cause (CC) 与 Common Effect (CE) 两种因果结构，且已有成熟的人类基线。作者认为，现在才值得做这件事，是因为前沿 LLM/VLM 在常规 reasoning benchmark 上已经追平甚至超过人类，于是"它们能不能像人一样迁移因果骨架"这个问题第一次变得可证伪，而不是被表面分数掩盖。论文实际影响范围有限——它不是提出新模型，而是给出"grounding 依赖"这一可量化的失败模式，提醒不要把 LLM 在纯文本 reasoning 上的表现外推为具备与人类对等的因果抽象。

核心观点

  • 当前 SOTA LLM/VLM 在 OpenLock 上表现出延迟或缺失的因果迁移：必须先做 environmental grounding，效率收益才出现。
  • 人类从第一次尝试就能用上先前的 CC/CE 结构知识，跨环境迁移几乎无额外代价。
  • 纯文本条件下模型发现效率能匹敌甚至超过人类；加入视觉（image-only 或 text+image）整体反而变差。
  • 模型在 CC vs CE 之间存在系统性不对称，而人类没有，暗示是启发式偏好而非方向中立的因果抽象。
  • 大规模统计学习不会自动产出人类那种脱离具体环境的因果 schema；grounding 依赖是 LLM/VLM 的根本性局限。

方法

沿用 OpenLock 范式：七根 lever + 一扇门的虚拟环境，agent 必须通过顺序交互发现隐藏的因果依赖，环境间 lever 位置/颜色/标签变，但底层结构（CC 或 CE）不变。实验分两阶段：Experiment 1（Baseline）在单一环境内测发现效率；Experiment 2（Transfer）把 agent 迁移到保留同一因果结构但外观改变的新环境，测"结构迁移带来的加速"。被试者涵盖 LLM/VLM（含 Claude-4.5）与人类参与者；模型按输入模态拆成 text-only (T)、image-only、text+image 三个条件。主要度量是 marginal discovery cost——找到第 k 个解所需的尝试数，以及 Experiment 1→2 的 efficiency gain ratio。

实验

环境：OpenLock，包含 CC 与 CE 两种因果结构。基线：人类参与者作为金标准，辅以不同输入模态下的模型条件对照。指标：marginal discovery cost（每个顺序解的尝试数）和 (Attempts_base − Attempts_trans) / Attempts_base 的 efficiency gain ratio。Figure 1 给出环境布局与两种因果结构的示意。

上图展示的是实验台本身：左边是带 7 根 lever + 1 扇门的 OpenLock 虚拟环境，右边是 CC 与 CE 两种底层因果结构的示意。它支撑的是"顺序发现因果依赖 + 跨环境结构不变、外观变化"这一实验设置——没有它，后文所有 marginal discovery cost 和 transfer gain 的对比都无从定义。

结果

在 text-only 条件下模型的发现效率追平甚至超过人类；image-only 和 text+image 条件整体拖累表现，说明模型主要靠符号处理而非真正的多模态整合。CC/CE 之间人类表现大致对称，模型则系统性偏向某一方向。

这张图支撑"纯文本条件下模型能追平人类发现效率"这一主张：分别画出 CC（左）和 CE（右）下，每个后续解所需的尝试数曲线，人类与 Claude-4.5 等模型（T 条件）放在一起比较，可见文本条件下模型曲线贴近甚至压过人类曲线，但 CC/CE 两侧形状不对称。

这张柱状图是论文核心结论的定量证据：Experiment 1→2 的 efficiency gain ratio，即从 baseline 迁到新环境后尝试数下降的比例。人类柱子明显高，说明迁移立刻兑现；多数模型柱子低甚至接近 0，证明它们没有从结构迁移中获得收益。

这张图把 baseline（灰）与 transfer（红）的 marginal discovery cost 叠在一起，逐模型和人类对比。人类的红色曲线从第一次尝试就显著低于灰色，体现"first-attempt 就用上先验结构"；模型的红灰曲线几乎重合，只有在后续尝试才慢慢分开，正好对应作者所说的 environmental grounding 必须先行。

结论

实践者该带走的一点是：当前 LLM/VLM 的"因果迁移"其实是 grounding 后的局部加速，而不是人类式的结构复用——在同一物理布局里它可以追平甚至超过人类（text-only 条件），但换个外观就要重新摸索，transfer gain ratio 明显低于人类。边界也要讲清楚：结论建立在 OpenLock 这一单一人机对比范式、CC/CE 两种结构和包括 Claude-4.5 在内的若干前沿模型上，没覆盖更复杂的因果图、真实物理交互或经过显式 causal-tuning 的 agent；视觉条件的退化也可能部分来自 VLM 对这类抽象示意图的感知瓶颈，而非纯粹的因果抽象缺陷。

是否新瓶装旧酒

这是偏诊断性的新工作，不是换皮。相关前置研究：

  • 原始 OpenLock + 经典 RL agent 的人机对比研究，已指出 RL 在此任务上灾难性失败——本文把被测对象换成前沿 LLM/VLM，并明确提出 “environmental grounding” 这一中间阶段作为新机制描述，不只是换 benchmark 刷榜。
  • LLM causal reasoning 评测（如各类 causal QA / counterfactual benchmark）大多是静态文本问答，本文强调交互式顺序发现 + 结构迁移，设定不同。
  • VLM 多模态推理评测常报告"加视觉就更好"；本文反过来给出视觉条件整体拖累表现的反例，是一个有价值的负面发现。 增量部分：CC/CE 不对称这一观察偏启发式，尚未给出机制解释。总体判断是真工作，但价值在"划出失败模式"而非提出新方法。

尚未回答的问题

  • grounding 阶段具体在模型内部发生了什么？是 in-context 记忆，还是某种 latent schema 形成？
  • 经过显式 causal fine-tuning 或工具使用（如调用符号求解器）的 agent 是否仍表现出同样的 grounding 依赖？
  • 视觉条件下的退化有多少来自感知瓶颈、多少来自推理瓶颈？用更清晰的结构化视觉输入能否翻转？
  • CC/CE 不对称的方向与强度是否随模型规模 / 家族系统变化？
  • 在更复杂、非二元的因果图或真实机器人交互中，这种 grounding-dependent transfer 还成立吗？

原始摘要（中文翻译）

从具体情境中抽取抽象的因果结构并将其应用到新情境，是人类智能的标志之一。尽管大语言模型（LLMs）和视觉语言模型（VLMs）在广泛的推理任务上表现出色，它们进行交互式因果学习——通过顺序探索归纳潜在结构并跨情境迁移——的能力仍未被刻画。人类学习者在极少暴露后就能完成此类迁移，而经典强化学习（RL）agent 则灾难性失败。当前最先进的 AI 模型是否具备类人的抽象因果结构迁移机制，仍是一个开放问题。我们使用 OpenLock 范式（要求顺序发现 Common Cause (CC) 与 Common Effect (CE) 两类结构）来研究这一问题，结果表明：模型表现出根本上延迟或缺失的迁移能力——即便是成功的模型，也必须先进行环境特定的映射（我们称之为 environmental grounding），效率收益才会出现；而人类从第一次求解尝试起就能利用先验结构知识。在 text-only 条件下，模型的发现效率匹敌甚至超过人类；与之相反，无论是 image-only 还是 text-and-image 条件下的视觉信息，整体上都降低而非提升了表现，暴露出模型广泛依赖符号处理，而非真正整合多模态推理。模型还表现出人类所没有的系统性 CC/CE 不对称，暗示其使用的是启发式偏好而非方向中立的因果抽象。这些发现揭示：大规模统计学习不会产出支撑人类类比推理的去情境化因果 schema，并将 grounding 依赖的迁移确立为当前 LLM 与 VLM 的一项根本性局限。
